<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Yao允许开发人员通过流程创建Web服务。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Yao 是一个应用程序引擎，它可以创建数据库模型，编写 API 服务，并仅通过 JSON 描述用于 Web 和硬件的仪表板界面，并具有 10 倍的生产力。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Yao基于</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">流式</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">编程思想，采用</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Go</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">语言开发，支持多种方式扩展数据流处理器。</font><font style="vertical-align: inherit;">这使得Yao极其通用，可以在大多数场景下替代编程语言，并且在可重用性和编码效率方面比传统编程语言提高10倍；</font><font style="vertical-align: inherit;">应用性能和资源利用率优于</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">PHP</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">JAVA</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">等语言。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Yao有一个内置的数据管理系统。</font><font style="vertical-align: inherit;">通过编写</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">JSON</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">来描述界面布局，可以实现90%的常见界面交互功能。</font><font style="vertical-align: inherit;">特别适合快速制作各种管理后台、CRM、ERP等企业内部系统。</font><font style="vertical-align: inherit;">特殊的交互功能还可以通过编写扩展组件或HTML页面来实现。</font><font style="vertical-align: inherit;">内置的管理系统不与Yao耦合，可以使用</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">VUE</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">React</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">等任何前端技术来实现管理界面。</font></font></p>
